### Design Decisions

| Decision | Why |
|---|---|
| **No animation libraries** | All motion is pure CSS transitions + Tailwind utilities вҖ” zero JS bundle overhead |
| **No icon libraries** | Every icon is an inline SVG вҖ” no FontAwesome, no Lucide, no extra HTTP requests |
| **Modular section directories** | Each section owns its constants, types, and sub-components вҖ” easy to modify or remove |
| **`"use client"` explicit** | Only interactive components opt into client rendering вҖ” server components by default |
| **IntersectionObserver for everything** | Scroll-triggered animations + nav highlighting without scroll event listeners |
