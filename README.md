## ⚡ What is Neo?

> *"It's a portfolio that looks like it was designed by a developer who actually enjoys design."*

**Neo** is a personal developer portfolio built around the **Neo-Brutalism** design movement. Not your average clean-minimal-white-space portfolio, this one has **personality**.

<table>
<tr>
<td width="50%">

### ✦ The Philosophy

```
┌─────────────────────────────────┐
│                                 │
│  Neo-Brutalism ≠ Minimalism    │
│  Neo-Brutalism = Maximalism    │
│                 with Discipline │
│                                 │
│  ┌──────────┐                   │
│  │  BOLD    │ ← 4px borders    │
│  │  DESIGN  │ ← flat shadows   │
│  └──────────┘ ← zero blur      │
│       ↗                         │
│  shadow: 6px 6px 0px #000       │
│                                 │
└─────────────────────────────────┘
```

</td>
<td width="50%">

| | Principle | Detail |
|---|---|---|
| 🟨 | **Thick Borders** | `4px solid black` everywhere |
| ⬛ | **Flat Shadows** | Offset shadows, zero blur |
| 🟪 | **Bold Colors** | Violet, pink, yellow, emerald |
| 🔤 | **Raw Typography** | Monospace code + uppercase |

The landing section is a **freeform canvas layout** on desktop — not a rigid grid. Elements are absolutely positioned to create an editorial, magazine-style spread:

<br>

<div align="center">

| Position | Component | Interaction |
|:---:|---|---|
| ↖️ **Top-Left** | `WhoamiCard` — terminal `$ whoami` | Click to cycle roles: `Developer` → `Student` → `Problem Solver` → `Tech Enthusiast` |
| ⬆️ **Top-Center** | `Hero` — "Welcome To My Portfolio" | Click the main banner to cycle through 5 accent background colors |
| ↗️ **Top-Right** | `CommentCard` — code comment block | Click to rotate through 3 developer philosophy quotes |
| ↙️ **Bottom-Left** | `CodeWindow` — faux code editor | Click the language badge to cycle: `TS` → `JS` → `PY` → `GO` |
| ↘️ **Bottom-Right** | `TerminalSteps` — build terminal + staircase | Click terminal for build logs, click stairs to change color |

```
  │    ████  #06b6d4   Cyan        ← Cool accent                   │
  │    ████  #000000   Black       ← Borders, shadows, text        │
  │    ████  #f9f8f6   Off-White   ← Background base               │
  │                                                                 │
  └─────────────────────────────────────────────────────────────────┘
```

### Typography

| Font | Variable | Usage |
|---|---|---|
| **Geist Sans** | `--font-geist-sans` | Headings, navigation, body text |
| **Geist Mono** | `--font-geist-mono` | Code blocks, terminal cards, tags, badges |

### Interactive Color Cycling

Almost every decorative element is **clickable** and cycles through curated palettes:

```typescript
// Every click reveals a new color — making the portfolio explorable
const portfolioBgs = ["bg-yellow-400", "bg-[#fa5b8d]", "bg-[#8b5cf6]", "bg-orange-400", "bg-emerald-400"];
const [index, setIndex] = useState(0);

const handleClick = () => setIndex((prev) => (prev + 1) % portfolioBgs.length);
```

### Background

```css
/* Graph-paper grid overlay — the blueprint aesthetic */
body {
  background: #f9f8f6;
  background-image:
    linear-gradient(to right, rgba(0,0,0,0.07) 1px, transparent 20px),
    linear-gradient(to bottom, rgba(0,0,0,0.07) 1px, transparent 2px);
  background-size: 20px 20px;
}
```

## 📐 Architecture

<br>

```
neo/
│
├── 📄 app/
│   ├── layout.tsx              # Root layout — Geist fonts, metadata, favicon
│   ├── page.tsx                # Home — assembles all 6 sections
│   ├── globals.css             # Graph-paper grid + Tailwind theme tokens
│   └── icon.png                # Favicon + Apple touch icon
│
├── 🧩 components/
│   │
│   │  ── Hero Canvas ──────────────────────────────────────
│   ├── Navbar.tsx              # Sticky nav · brutalist box · IntersectionObserver
│   ├── Hero.tsx                # "Welcome To My Portfolio" · color cycling
│   ├── CodeWindow.tsx          # Faux code editor · language badge toggle
│   ├── WhoamiCard.tsx          # $ whoami terminal · role cycling
│   ├── CommentCard.tsx         # /* comment */ block · rotating quotes
│   ├── TerminalSteps.tsx       # npm run dev terminal · staircase SVG
│   ├── HomeMobile.tsx          # Mobile-specific hero layout
│   │
│   │  ── Projects ─────────────────────────────────────────
│   ├── Projects.tsx            # 3×3 grid · code doodle margins
│   ├── ProjectCard.tsx         # Reusable card · staggered animations
│   │
│   │  ── Section Modules ──────────────────────────────────
│   ├── 📁 about/              # 9 files — bio, socials, doodles, stairs
│   ├── 📁 contributions/      # 9 files — GitHub heatmap, LeetCode stats
│   ├── 📁 skills/             # 7 files — 5 categories, 30+ tech badges
│   └── 📁 contact/            # 6 files — cards, footer, constants
│
└── 🖼️ public/
    ├── icon.png                # Brand icon
    └── springboot-icon.svg     # Custom Spring Boot icon
```

### Design Decisions

| Decision | Why |
|---|---|
| **No animation libraries** | All motion is pure CSS transitions + Tailwind utilities — zero JS bundle overhead |
| **No icon libraries** | Every icon is an inline SVG — no FontAwesome, no Lucide, no extra HTTP requests |
| **Modular section directories** | Each section owns its constants, types, and sub-components — easy to modify or remove |
| **`"use client"` explicit** | Only interactive components opt into client rendering — server components by default |
| **IntersectionObserver for everything** | Scroll-triggered animations + nav highlighting without scroll event listeners |

## 🚀 Quick Start

<br>

### Prerequisites

```
Node.js  ≥ 18.17
npm / yarn / pnpm / bun
```

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/Viidhyanshu/Neo-Portfolio.git

# 2. Navigate into the project
cd Neo-Portfolio

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

```bash
npm start          # Start production server
```
